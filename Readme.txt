■このゲームについて
舞台は赤ちゃんの夢の中...おもちゃの積み木は不気味に動きだし襲い掛かってきた。Voxmare は相手の弾を跳ね返して戦うシューティングゲームです。ボスの形状は遊ぶたびに変わり、あなたがどこを壊すかによって常にその姿を変えます。様々なおもちゃの人形を仲間にしながらボスのコアを全て破壊しよう！

In a baby's dream... Toy blocks began to move eerily and attacked. Voxmare is a shooting game in which you fight by reflecting enemy's bullets. The boss changes shape each time you play, and it always changes their appearance depending on which parts you break. Get various toy dolls and destroy all the boss's cores!

■起動方法
「Voxmare.exe」を実行してください。

■基本操作(デフォルト)
ゲーム開始時のチュートリアル画面または、ゲーム配布ページのスクリーンショットをご覧ください。

■Q&A
Q.相手が弾を撃ってこない。画面の表示がおかしい。
A.Escキーでポーズ画面を開き、メインメニューに戻り、再度レベル選択でゲームを開始すると改善できる場合があります。
